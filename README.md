# JimBot

## A versatile Python Discord bot with music, minigames, and more!

**Features:**

- Music playback powered by Lavalink
- Fun minigames to engage users
- Informative commands for various purposes
- Hosted on Raspberry Pi 3B for resource efficiency

**Technology Stack:**

- Python 3.10.5
- discord.py API
- Raspbian GNU/Linux 11 (bullseye) operating system

**Installation and Setup:**

1. Clone the repository:
   git clone [https://github.com/snof26/jimbot.git](https://github.com/snof26/jimbot.git)

2. Activate the Python environment:
   `source env/bin/activate`

3. Install dependencies:
   `pip install -r requirements.txt`

**Running the Bot:**

Start the main bot process:
`python3 bot.py`
Start Lavalink for music functionality (in a separate terminal):
`cd cogs`
`java -jar Lavalink.jar`
